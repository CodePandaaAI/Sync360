# Code of Conduct

## Our pledge

We want Sync360 to be a welcoming project for people learning, building, testing, and discussing local-network software.

Everyone participating in this project is expected to be respectful, patient, and constructive.

## Expected behavior

- Be kind and direct.
- Assume good intent when possible.
- Give feedback on code and ideas, not on people.
- Be patient with beginners and people learning in public.
- Keep discussions focused and useful.
- Respect different experience levels and backgrounds.

## Unacceptable behavior

- Harassment, insults, threats, or personal attacks.
- Dismissive or mocking comments about someone learning.
- Sexualized language or imagery.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeated off-topic disruption.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or block participants who violate this code of conduct.

If you see a problem, report it privately to the maintainer.
